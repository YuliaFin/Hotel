Для начала процесса бронирования номера в подсистеме необходимо перейти на главную страницу и найти раздел «Войти». В этом разделе необходимо ввести учетные данные, такие как логин и пароль.
После ввода данных нажать на кнопку «Войти» для выполнения входа в подсистему.
После успешного входа в систему, пользователь будет перенаправлен на главную страницу, где следует найти раздел «Номера». В этом разделе предоставляется возможность указать дату заезда и дату выезда, чтобы найти свободные номера в заданный период. После нажатия на кнопку «Найти свободные номера», подсистема выполнит поиск доступных номеров, соответствующих заданным критериям.
Пользователь получает список номеров, которые свободны на выбранные даты.
Из предложенного списка номеров необходимо выбрать подходящий и нажать на кнопку «Забронировать».
После этого откроется форма бронирования номера, где нужно заполнить обязательные поля, такие как даты заезда и выезда, количество основных и дополнительных мест.
Рекомендуется проверить правильность введенных данных и нажать на кнопку "Создать". После выполнения всех указанных действий, система создаст заявку на бронирование номера.
Далее пользователь может отследить изменение статуса заявки, перейдя в раздел «Заявки на бронирование». Там находится информация о всех заявках, которые пользователь отправил и есть возможность отслеживать статусы заявок.
Следуя этим указаниям, пользователь сможет успешно бронировать номера в подсистеме.
